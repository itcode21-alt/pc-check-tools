# 내부용 AdSense 점검표

이 문서는 공개 페이지에 노출하지 않는 내부 확인용 메모입니다.

## 공개 전 확인

- 개인정보처리방침, 이용약관, 소개, 문의 페이지가 공개되어 있는지 확인
- 메인 페이지에서 진단과 가이드가 바로 보이는지 확인
- 각 상세 페이지가 충분한 본문과 FAQ를 포함하는지 확인
- 모바일에서 줄바꿈, 카드 높이, 버튼 간격이 깨지지 않는지 확인
- 깨진 링크, 중복 제목, 비어 있는 섹션이 없는지 확인

## AdSense 제출 전 확인

- 콘텐츠가 거의 없는 페이지는 광고 대상에서 제외
- 탐색 메뉴가 직관적이고 방문자가 길을 잃지 않는지 확인
- 개인정보처리방침에 쿠키, 광고 서비스, 데이터 수집과 사용 범위가 적혀 있는지 확인
- 사이트가 단순 링크 모음이 아니라 실제 정보 페이지로 보이는지 확인
- 오해를 부르는 표현이나 과장된 보장 문구가 없는지 확인

## 운영 메모

- 이 파일은 공개 메뉴나 사이트맵에 연결하지 않음
- 배포본에는 노출되지 않는 숨김 참고용으로만 유지
