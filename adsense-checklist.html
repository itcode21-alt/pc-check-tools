홈진단가이드

심사 전 마지막 확인

아래 항목은 사이트 전체가 정보 제공형으로 보이는지, 페이지가 충분한 내용을 갖췄는지, 그리고 방문자가 쉽게 이동할 수 있는지를 점검하기 위한 목록입니다.

콘텐츠 깊이

증상 상세 페이지가 단순 요약이 아니라 원인, 순서, 예외 상황까지 설명하는지 확인합니다.

페이지 완성도

소개, 개인정보처리방침, 이용약관, 문의 페이지가 존재하고 서로 연결되는지 확인합니다.

탐색성

메인, 진단, 가이드, 상세 페이지가 서로 오가기 쉽고 메뉴가 끊기지 않는지 봅니다.

체크 항목

광고보다 내용 우선

광고가 콘텐츠보다 먼저 보이거나 클릭을 유도하는 배치가 없는지 확인합니다.

정확한 안내

쿠키, 외부 링크, 사용자 입력 데이터 처리 방식이 개인정보처리방침에 반영되어 있는지 확인합니다.

충분한 분량

8개 증상 상세 페이지가 각각 독립적인 글처럼 읽히도록 길이와 구조가 확보되어 있는지 확인합니다.

반복 최소화

같은 문장과 같은 카드 구조가 너무 많아 보이지 않도록 페이지별로 차이를 두었는지 확인합니다.

모바일 대응

좁은 화면에서도 카드와 버튼이 줄바꿈되며, 내용이 화면 밖으로 넘치지 않는지 확인합니다.

제목과 설명

각 페이지의 title과 description이 서로 다르고, 실제 내용과 어긋나지 않는지 확인합니다.
