WINDOWS NETWORK BASICS

IP를 받아도 인터넷이 안 되면, 연결 단계부터 나눠 확인하세요

케이블 연결, 공유기 통신, DNS 이름 해석은 서로 다른 단계입니다. 무료 프로그램 설치보다 Windows 기본 기능으로 원인을 먼저 구분하세요.

1. 가장 먼저 확인할 것

  1. 다른 기기에서도 같은 인터넷이 되는지 확인합니다. 다른 기기도 안 되면 PC보다 공유기·회선 문제일 가능성이 큽니다.
  2. 유선은 LAN 포트의 링크 LED와 케이블을 확인하고, 가능하면 다른 포트·케이블로 바꿉니다.
  3. Windows 설정 → 네트워크 및 인터넷에서 비행기 모드, VPN, 프록시가 켜져 있지 않은지 확인합니다.

2. IP·DNS 확인하기

  1. 시작 메뉴에서 명령 프롬프트를 열고 ipconfig를 입력합니다.
  2. 사용 중인 이더넷 또는 Wi-Fi에 IPv4 주소와 기본 게이트웨이가 있는지 확인합니다.
  3. ping 8.8.8.8은 되지만 웹사이트가 안 열리면 DNS 문제일 수 있습니다. ipconfig /flushdns를 실행한 뒤 다시 확인합니다.

3. 어댑터·드라이버 점검

장치 관리자 → 네트워크 어댑터에서 노란 느낌표가 있는지 확인합니다. 최근 업데이트 뒤부터 문제라면 드라이버를 롤백하고, 없다면 PC·메인보드 제조사의 공식 LAN/Wi-Fi 드라이버를 설치합니다. 자동 드라이버 설치 프로그램은 사용하지 마세요.

4. 마지막 수단: 네트워크 초기화

Windows 11은 설정 → 네트워크 및 인터넷 → 고급 네트워크 설정 → 네트워크 초기화에서 실행합니다. 저장된 Wi-Fi, VPN, 가상 어댑터가 제거될 수 있으므로 필요한 접속 정보는 미리 확인하세요. 초기화 뒤 재부팅이 필요합니다.

회사·학교 PC

고정 IP, 프록시, VPN을 사용하는 환경에서는 초기화 전에 관리자에게 설정값을 확인하세요. 임의 초기화로 업무망 접속 정보가 사라질 수 있습니다.
