📚 PC 초보자 완벽 가이드

PC를 처음 구매하는 분들을 위한 완벽 가이드입니다

1️⃣ PC 부품의 이해

🖥️ 중앙처리장치 (CPU)

CPU는 PC의 "뇌"입니다. 모든 명령을 처리하고 계산합니다.

선택 기준:
  • 메인 사용 목적에 따라 결정
  • 코어 수가 많을수록 멀티태스킹이 좋음
  • 클록 속도가 높을수록 빠름

⚙️ 메인보드 (Motherboard)

메인보드는 모든 부품을 연결하는 "중앙 신경계"입니다.

주의사항:
  • CPU와 반드시 호환되는 소켓 확인
  • RAM 슬롯 개수 확인 (향후 업그레이드용)
  • 포트 종류와 개수 확인

🎮 그래픽카드 (GPU)

화면에 보이는 모든 그래픽을 처리합니다.

구분:
• 내장 그래픽: CPU에 포함 (일반 사용 충분)
• 외장 그래픽: 게이밍/영상 작업 필요 (별도 구매)

⚡ 파워서플라이 (PSU)

PC에 전력을 공급합니다. 충분한 용량이 필수입니다.

용량 계산:
CPU TDP + GPU TDP × 1.3 = 권장 PSU
예: 65W + 200W = 265W → 350~400W PSU 추천

🧠 메모리 (RAM)

실행 중인 프로그램의 데이터를 일시 저장합니다.

용량 기준:
  • 일반 사용: 16GB
  • 게이밍/영상 편집: 32GB
  • 전문가용: 64GB 이상

💾 저장장치 (SSD/HDD)

데이터를 영구 저장합니다.

비교:
항목 SSD HDD
속도 매우 빠름 ⚡ 느림 🐌
안정성 우수 ✓ 낮음
가격 중간 💰 저렴 💵
추천 필수 보조 용도

2️⃣ 부품 호환성 확인

🔗 필수 호환성 체크

CPU ↔ 메인보드

소켓이 일치해야 함
예: LGA1700, AM5 등

GPU ↔ PSU

파워 용량이 충분해야 함
30~50% 여유 필수

RAM ↔ 메인보드

슬롯과 DDR 세대 일치
예: DDR5 LGA1700

SSD ↔ 메인보드

M.2 슬롯 필수
NVMe 권장

💡 팁: ITSVC의 "호환성 검사" 도구를 사용하면 쉽게 확인할 수 있습니다!

3️⃣ 현명한 구매 방법

🛒 구매처 선택

1

대형 유통점

다양한 선택지, 비교 검색 가능, 배송 빠름

2

전문 쇼핑몰

전문 상담 가능, 맞춤 조립 서비스, A/S 확실

3

비교 사이트

가격 비교 쉬움, 리뷰 확인 가능, 최저가 찾기 용이

💰 예산 배분

⚠️ 일반적인 예산 배분

• CPU: 20~30%
• GPU: 30~40% (게이밍 시)
• RAM: 10~15%
• SSD: 10~15%
• PSU/기타: 10~20%

⏱️ 구매 시기

좋은 타이밍:
  • 신제품 출시 후 2~3개월 (가격 안정화)
  • 명절 기간 (세일)
  • 구형 부품 재고 정리 시

4️⃣ PC 용어 쉽게 이해하기

TDP (Thermal Design Power)
부품이 발생하는 열의 양(와트 단위). PSU 선택에 중요합니다.
소켓 (Socket)
CPU를 메인보드에 연결하는 규격. 반드시 일치해야 합니다. (LGA1700, AM5 등)
클록 (Clock Speed)
CPU가 1초에 수행하는 작업 수(GHz 단위). 높을수록 빠릅니다.
코어 (Core)
CPU의 처리 단위. 많을수록 멀티태스킹이 좋습니다. (4코어, 8코어 등)
VRAM
그래픽카드의 메모리. 많을수록 고해상도/고주사율 플레이 가능. (2GB, 4GB, 8GB 등)
대역폭 (Bandwidth)
메모리나 SSD에서 초당 처리할 수 있는 데이터양. 높을수록 빠릅니다.
BIOS
메인보드의 기본 소프트웨어. 부품 업데이트나 설정 변경 시 필요합니다.

5️⃣ PC 구성 체크리스트

구매 전 확인사항

  • CPU와 메인보드의 소켓 일치 확인
  • 메인보드가 선택한 RAM 지원 확인
  • M.2 슬롯이 있는 메인보드 선택
  • PSU 용량이 충분한지 계산 (TDP × 1.3)
  • 그래픽카드 크기가 케이스에 맞는지 확인
  • CPU 쿨러가 필요한지 확인
  • 케이스 포트와 케이블 종류 확인
  • A/S 기간과 보증 조건 확인

구성 후 확인사항

  • 모든 부품이 제대로 설치됐는지 확인
  • BIOS에서 모든 부품 인식 확인
  • 온도가 정상 범위인지 확인 (CPU <70°C, GPU <80°C)
  • 성능 벤치마크 테스트 실행
  • 스트레스 테스트로 안정성 확인

6️⃣ 자주 묻는 질문

Q. 기존 부품을 재사용해도 되나요?

네, 호환되는 부품이라면 재사용 가능합니다. 다만 오래된 부품은 성능 저하가 있을 수 있습니다. 특히 파워(PSU)는 안정성을 위해 새로운 제품을 권장합니다.

Q. 조립이 어렵지 않을까요?

생각보다 간단합니다. 대부분의 부품은 제자리에 "끼워지도록" 설계되었습니다. 유튜브 조립 영상을 참고하거나 전문점의 조립 서비스를 이용할 수 있습니다.

Q. A/S는 어떻게 받나요?

각 부품마다 제조사별 A/S를 받습니다. 구매 영수증과 등록증을 보관하세요. 대형 유통점에서 구매하면 초기 불량 교환이 더 쉽습니다.

Q. 부품을 업그레이드할 수 있나요?

네, RAM, SSD, 그래픽카드는 쉽게 업그레이드할 수 있습니다. CPU와 메인보드 업그레이드는 전문 지식이 필요할 수 있습니다.
