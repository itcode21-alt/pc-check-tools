BEGINNER GUIDE

PC를 처음 구매하는 분을 위한 가이드

부품 이름이 낯설어도 괜찮습니다. 각 부품의 역할과 선택 기준, 호환성 확인 순서를 단계적으로 안내합니다.

PC 부품의 이해

중앙처리장치 (CPU)

PC의 "뇌"입니다. 모든 명령을 처리하고 계산합니다. 코어 수가 많을수록 멀티태스킹이 좋고, 클록 속도가 높을수록 단일 작업이 빠릅니다.

선택 기준
  • 메인 사용 목적(사무·게임·영상 편집)에 따라 결정
  • 코어 수가 많을수록 멀티태스킹에 유리
  • 클록 속도가 높을수록 단일 처리가 빠름

메인보드

모든 부품을 연결하는 "중앙 신경계"입니다. CPU와 반드시 소켓이 일치해야 하며, 나머지 부품과의 호환성도 메인보드에서 결정됩니다.

주의사항
  • CPU 소켓 규격 일치 필수 (LGA1700, AM5 등)
  • RAM 슬롯 개수 확인 — 향후 증설 여유 고려
  • M.2 슬롯 여부와 포트 종류 확인

그래픽카드 (GPU)

화면에 보이는 그래픽을 처리합니다. 일반 사무 용도는 CPU 내장 그래픽으로 충분하고, 게임이나 영상 편집에는 별도 외장 그래픽이 필요합니다.

  • 내장 그래픽 — CPU에 포함, 일반 사용에 충분
  • 외장 그래픽 — 게이밍·영상 작업 시 별도 구매

파워서플라이 (PSU)

PC에 전력을 공급합니다. 용량이 부족하면 전원이 켜지지 않거나 게임 중 재부팅이 발생합니다.

간단 계산식
CPU TDP + GPU TDP × 1.3 = 권장 PSU 용량
예: 65W + 200W × 1.3 ≈ 345W → 400W 이상 선택

메모리 (RAM)

실행 중인 프로그램의 데이터를 임시로 저장합니다. 용량이 부족하면 디스크 사용률이 높아지고 속도가 크게 떨어집니다.

  • 일반 사무·인터넷 — 16 GB
  • 게이밍·영상 편집 — 32 GB
  • 전문가용 작업 — 64 GB 이상

저장장치 (SSD / HDD)

데이터를 영구적으로 저장합니다. 운영체제와 자주 쓰는 프로그램은 SSD에, 대용량 파일은 HDD에 저장하는 방식이 일반적입니다.

항목SSDHDD
속도매우 빠름느림
내구성우수 (충격에 강함)기계식, 충격에 약함
가격용량 대비 중간용량 대비 저렴
권장 용도OS·프로그램 (필수)대용량 보조 저장

부품 호환성 확인

부품을 구매하기 전에 반드시 아래 네 가지 조합이 맞는지 확인하세요. 하나라도 맞지 않으면 조립이 되지 않거나 성능이 제한됩니다.

CPU ↔ 메인보드

소켓 규격이 반드시 일치해야 합니다.
예: Intel LGA1700, AMD AM5

GPU ↔ PSU

파워 용량이 충분해야 하고, 보조전원 커넥터 규격도 확인하세요.

RAM ↔ 메인보드

DDR 세대(DDR4·DDR5)와 슬롯 규격이 일치해야 합니다.

SSD ↔ 메인보드

M.2 슬롯 유무와 NVMe 지원 여부를 확인하세요.

빠른 확인 방법 — 호환성 검사 도구에서 선택한 부품 조합을 검증할 수 있습니다.

현명한 구매 방법

구매처 선택

  1. 대형 유통 쇼핑몰 — 다양한 선택지, 가격 비교 쉬움, 배송 빠름
  2. 전문 PC 쇼핑몰 — 전문 상담 가능, 맞춤 조립 서비스, A/S 확실
  3. 가격 비교 사이트 — 최저가 탐색, 구매 후기 확인 용이

예산 배분 기준

일반적인 예산 비율
  • CPU: 20~30%
  • GPU: 30~40% (게이밍 시)
  • RAM: 10~15%
  • SSD: 10~15%
  • PSU·케이스·기타: 10~20%

구매 적기

  • 신제품 출시 후 2~3개월 — 가격 안정화 시점
  • 명절·연말 세일 기간
  • 구형 부품 재고 정리 시 — 이전 세대 가성비 제품 노릴 때

PC 용어 쉽게 이해하기

TDP

부품이 발생시키는 열의 양(와트 단위). PSU 용량과 쿨러 선택 기준이 됩니다.

소켓 (Socket)

CPU와 메인보드 연결 규격. LGA1700, AM5처럼 반드시 일치해야 합니다.

클록 속도

CPU가 1초에 처리하는 작업 수(GHz 단위). 높을수록 단일 작업이 빠릅니다.

코어 (Core)

CPU의 처리 단위. 4코어·8코어처럼 많을수록 멀티태스킹에 유리합니다.

VRAM

그래픽카드의 전용 메모리. 고해상도·고주사율 게임일수록 많이 필요합니다.

BIOS / UEFI

메인보드에 내장된 기본 소프트웨어. 부팅 순서와 부품 설정을 제어합니다.

NVMe / SATA

M.2 SSD의 인터페이스 종류. NVMe가 훨씬 빠르며 최신 보드 대부분 지원합니다.

DDR4 / DDR5

RAM 세대 규격. 메인보드가 지원하는 세대와 반드시 일치해야 합니다.

PC 구성 체크리스트

구매 전 확인사항

  • CPU와 메인보드의 소켓 규격 일치
  • 메인보드가 선택한 RAM DDR 세대 지원
  • M.2 슬롯이 있는 메인보드 선택
  • PSU 용량이 충분한지 계산 (TDP 합산 × 1.3)
  • 그래픽카드 크기가 케이스에 맞는지 확인
  • CPU 번들 쿨러 포함 여부 또는 별도 쿨러 필요 여부
  • 케이스 포트와 메인보드 헤더 연결 가능 여부
  • A/S 기간과 보증 조건 확인

구성 후 확인사항

  • BIOS에서 모든 부품이 인식되는지 확인
  • CPU 온도 정상 범위 확인 (부하 시 80°C 이하 권장)
  • GPU 온도 정상 범위 확인 (부하 시 85°C 이하 권장)
  • RAM 용량이 설치한 것과 일치하는지 확인
  • SSD가 올바른 속도로 인식되는지 확인 (NVMe 여부)

자주 묻는 질문

기존 부품을 재사용해도 되나요?

호환되는 부품이라면 재사용 가능합니다. 다만 오래된 PSU는 안전성을 위해 교체를 권장합니다. CPU나 메인보드를 새로 바꾸면 RAM 규격도 달라질 수 있으니 함께 확인하세요.

직접 조립이 어렵지 않나요?

대부분의 부품은 맞는 자리에만 끼워지도록 설계돼 있어 생각보다 간단합니다. 제조사 공식 설치 영상이나 유튜브 조립 가이드를 참고하면 됩니다. 자신 없다면 전문점의 조립 서비스를 이용하세요.

A/S는 어떻게 받나요?

각 부품마다 제조사 A/S를 개별로 받습니다. 구매 영수증과 시리얼 번호를 보관해 두세요. 대형 유통점에서 구매하면 초기 불량 교환이 더 수월합니다.

RAM이나 SSD를 나중에 업그레이드할 수 있나요?

RAM과 SSD는 비교적 쉽게 업그레이드할 수 있습니다. 단, 노트북은 RAM이 납땜된 경우가 많아 RAM 증설 가능 여부 확인을 먼저 해보세요. CPU와 메인보드 교체는 전체 구성을 바꾸는 수준이라 신중하게 판단이 필요합니다.

다음 단계 — 부품을 정했다면 호환성 검사 도구로 조합을 확인하고, PC 조립 후에는 PC 진단 도구로 이상 여부를 점검해 보세요.
