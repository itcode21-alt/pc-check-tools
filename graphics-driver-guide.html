무조건 최신 버전보다, 증상 전후의 변경을 먼저 확인하세요

화면 깨짐·깜빡임·게임 중 종료는 드라이버, 온도, 전원 문제 모두에서 생길 수 있습니다. 먼저 롤백을 시도하고, 반복될 때만 무료 DDU로 깨끗하게 재설치합니다.

시작 전 준비

그래픽카드 제조사 공식 사이트에서 설치할 드라이버를 먼저 내려받고, 시스템 복원 지점을 만드세요. 자동 드라이버 업데이트 프로그램은 사용하지 않습니다.

1. 최근 업데이트 뒤 시작됐다면: 이전 드라이버로 롤백

  1. 시작 메뉴에서 장치 관리자를 검색해 엽니다.
  2. 디스플레이 어댑터 → 사용 중인 그래픽카드 더블클릭 → 드라이버 탭을 엽니다.
  3. 드라이버 롤백이 활성화되어 있으면 선택한 뒤 재부팅합니다.

2. 반복될 때: DDU로 클린 재설치

  1. 공식 Wagnardsoft에서 무료 Display Driver Uninstaller(DDU)를 받습니다.
  2. 인터넷을 잠시 끊고, 안전 모드로 부팅합니다.
  3. DDU에서 GPU 제조사(NVIDIA·AMD·Intel)를 정확히 선택하고 제거 후 재시작을 실행합니다.
  4. 일반 모드로 돌아온 뒤 미리 내려받은 제조사 공식 드라이버를 설치합니다.

3. 드라이버가 아닌지 함께 확인

  • HWiNFO Free로 게임 중 GPU 온도와 전력 제한 여부를 확인합니다.
  • 그래픽카드 보조전원 케이블을 끝까지 다시 연결하고, 분배 케이블 대신 가능한 별도 케이블 연결을 확인합니다.
  • GPU·CPU 오버클럭과 언더볼트는 기본값으로 되돌린 뒤 재현 여부를 봅니다.
  • 다른 게임이나 벤치마크에서도 같으면 게임 단독 문제가 아닐 수 있습니다.
