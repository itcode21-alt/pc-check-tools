에러 코드 상세

코드 1 · 이 장치는 올바르게 작동하지 않습니다

장치 관리자에서 가장 일반적으로 보이는 오류입니다. 하드웨어가 있지만 제대로 작동하지 않는 상태를 나타냅니다. 오디오, 네트워크, 프린터 등 어떤 장치에서든 나타날 수 있습니다.

이 코드를 어떻게 해석해야 하나요?

코드 1은 가장 포괄적인 오류로, Windows가 장치를 찾은 뒤 드라이버로 제어하려고 했지만 실패했을 때 표시됩니다. 구체적인 이유를 알아내려면 장치 이름, 드라이버 버전, 최근 변경 사항을 함께 살펴야 합니다.

먼저 기록할 내용: 어느 장치에서 이 오류가 발생하는지, 언제부터인지, 최근 업데이트나 드라이버 설치 변경이 있었는지를 확인하세요.

가능성 높은 원인

  • 드라이버가 설치되지 않았거나 손상됨 (가장 일반적)
  • 드라이버 버전이 현재 Windows와 호환되지 않음
  • 하드웨어 전원이 공급되지 않음 (특히 USB 장치)
  • BIOS에서 장치 기능이 비활성화됨 (온보드 오디오, 카메라 등)
  • 하드웨어 자체의 물리적 불안정이나 접촉 불량

첫 점검 항목

  1. 장치 이름과 제조사 확인 — 장치를 마우스 오른쪽으로 클릭해 "속성"에서 정확한 모델명을 확인합니다.
  2. 드라이버 재설치 — 장치를 마우스 오른쪽으로 클릭하고 "드라이버 업데이트"를 선택합니다. "컴퓨터에서 드라이버 찾기"를 선택해 자동 검색시킵니다.
  3. 제조사 공식 드라이버 다운로드 — PC 또는 부품 제조사 웹사이트에서 정확한 모델용 최신 드라이버를 다운로드해 설치합니다.
  4. USB 장치라면 포트 변경 — 다른 USB 포트나 직접 연결을 시도합니다. 허브를 통하지 말고 PC에 직접 연결해보세요.
  5. BIOS에서 해당 장치 활성화 여부 확인 — 온보드 오디오, 카메라, 블루투스 등 내장 장치는 BIOS에서 비활성화되었을 수도 있습니다.

드라이버 완전 제거 후 재설치

위 방법으로 해결되지 않으면, 장치를 마우스 오른쪽으로 클릭해 "장치 제거"를 선택하고 "이 장치의 드라이버 소프트웨어 삭제"에 체크한 후 제거합니다. 그 후 PC를 재시작하면 Windows가 장치를 다시 감지합니다. 그다음 공식 드라이버를 설치합니다.

USB 장치 특별 경고

USB 주변 기기(마우스, 키보드, 프린터)에서 이 오류가 계속 나면:

  • 다른 USB 포트에 연결해보기
  • 다른 USB 케이블 사용해보기 (케이블이 손상되었을 수 있음)
  • 다른 PC에서 인식되는지 확인 (장치 자체 고장인지 PC 문제인지 구분)

데이터와 시스템을 보호하는 주의사항

드라이버를 제거하거나 BIOS를 수정하기 전에 중요한 파일을 백업해두세요. Windows 재설치가 필요할 수도 있습니다.

앞 단계로 해결되지 않을 때

모든 방법을 시도했는데도 코드 1이 계속 나타나면:

  • 다른 PC에서 같은 장치가 정상 작동하는지 확인
  • 그래도 작동하지 않으면 장치 제조사 지원에 문의
  • 온보드 장치(오디오, 카메라)라면 BIOS 재설정을 시도

Microsoft 지원: Device Manager error codes

자주 묻는 질문

  • 코드 1이 계속 나타나면 어떤 드라이버를 설치해야 하나요? 장치 이름(예: "Realtek High Definition Audio")을 장치 관리자에서 확인한 후, 그 정확한 이름으로 검색해 제조사 공식 사이트에서 드라이버를 다운로드하세요.
  • 자동 드라이버 검색은 왜 실패하나요? Windows의 자동 검색이 모든 드라이버를 찾지 못할 수 있습니다. 특히 구형 장치는 제조사 웹사이트에서 직접 다운로드해야 합니다.
