디아블로 4 오류 모음 — 원인과 해결 방법

디아블로 4는 상시 온라인 연결이 필요한 구조라, 로그인 실패를 반복 시도하는 것이 오히려 계정 잠금으로 이어지는 경우가 있어 주의가 필요합니다.

게임 실행 시 Fenris 오류 또는 응용 프로그램 오류로 실행 불가 · Fenris 오류 · 응용 프로그램 오류

실행/설치

디아블로 4를 실행하면 Fenris 오류나 응용 프로그램 오류가 발생하며 게임이 열리지 않는 경우입니다. 블리자드 공식 포럼에서 안내하는 대표적인 실행 실패 유형입니다.

가능한 원인

  • 그래픽카드 드라이버가 손상되었거나 최신 게임 버전과 호환되지 않는 경우
  • 게임 파일 일부가 손상된 경우
  • 그래픽 드라이버나 운영체제가 오래되어 호환성 문제가 발생하는 경우

점검 순서

  1. 1단계 - 그래픽 드라이버 클린 재설치 (공식 권장):
    ① DDU(Display Driver Uninstaller) 또는 AMD Cleanup Utility로 기존 드라이버를 완전히 삭제
    ② 그래픽카드 제조사 최신 드라이버 재설치
  2. 2단계 - 복구 도구 실행:
    ① Battle.net 앱에서 디아블로 4 선택 → 옵션 → 스캔 및 복구 실행으로 손상된 게임 파일 복구
  3. 3단계 - 드라이버·운영체제 업데이트:
    ① Windows 업데이트를 최신 상태로 유지
    ② 칩셋·그래픽 드라이버를 모두 최신 버전으로 업데이트
  4. 4단계 - 재설치:
    ① 위 방법이 모두 실패하면 게임을 완전히 삭제한 뒤 재설치

  2. 2단계 - 배틀넷 로그인 지역 변경:
    ① Battle.net 데스크톱 앱 로그인 화면에서 로그인 지역을 변경해보기
  3. 3단계 - 검사 및 복구:
    ① 배틀넷 앱에서 디아블로 IV 검사 및 복구 기능 실행

크로스플레이 중 파티초대·파티참가 버튼이 비활성화되는 문제 · 크로스플레이 파티초대 불가

접속/네트워크

플랫폼이 다른 친구와 크로스플레이를 시도할 때 파티초대·참가 버튼이 비활성화되거나, 한쪽에서만 상대가 오프라인으로 표시되는 경우입니다.

가능한 원인

  • 배틀넷 앱에 처리되지 않은 친구 요청이 쌓여 있는 경우
  • 크로스 네트워크 플레이 설정이 꺼져 있는 경우

점검 순서

  1. 1단계 - 보낸 친구 요청 정리:
    ① 배틀넷 앱 친구 설정에서 처리되지 않고 쌓인 보낸 친구 요청을 삭제
  2. 2단계 - 크로스 네트워크 플레이 설정 확인:
    ① 게임 내 설정 → 연결 → 크로스 네트워크 플레이가 켜져 있는지 확인
  3. 3단계 - 재초대 시도:
    ① 설정 변경 후 친구 목록을 새로고침하고 다시 파티 초대 시도

커뮤니티에서 확인된 사례

  • 커뮤니티 사례 보기 — 친구 추가(친추)가 안되는 문제 해결방법을 정리한 디아블로4 인벤 게시글.

백금화 등을 구매했는데 "처리 중"에서 멈추거나 지급되지 않는 문제 · 결제 처리 중 멈춤 · 상점 지급 오류

기타

Battle.net 샵에서 백금화나 아이템을 구매했지만 상점에서 계속 '처리 중'으로 표시되거나 실제로 지급되지 않는 경우입니다.

가능한 원인

  • 결제 처리 지연
  • 구매 후 게임 재시작을 하지 않아 반영되지 않은 경우

점검 순서

  1. 1단계 - 게임 재시작:
    ① 구매가 완료 상태로 표시된다면 게임을 재시작한 뒤 인게임 위치를 확인
  2. 2단계 - 구매 내역 확인:
    ① Battle.net 계정 페이지에서 실제 결제 완료 여부 확인
  3. 3단계 - 고객지원 문의:
    ① 결제는 완료됐는데 지급되지 않았다면 구매 내역과 함께 블리자드 고객지원에 문의
