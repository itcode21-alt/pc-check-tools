에러 코드 상세

0x80070102 · 네트워크 장치가 연결되지 않음

네트워크 어댑터나 케이블이 시스템에서 인식되지 않을 때 나타나는 오류로, 물리적 연결 문제나 드라이버 미인식이 주요 원인입니다.

이 코드를 어떻게 해석해야 하나요?

업데이트 관련 코드는 네트워크, 캐시, 구성 요소 저장소, 디스크 공간, 서비스 상태 중 어느 지점에서 작업이 중단됐는지를 보여줍니다. 실패가 반복되는 시점과 이벤트 로그의 세부 코드를 함께 확인하면 원인을 좁히기 쉽습니다.

먼저 기록할 내용: 0x80070102가 발생한 시각, 직전에 실행한 작업, 최근 설치한 드라이버·업데이트, 재부팅 뒤 같은 코드가 반복되는지를 적어 두세요.

가능성 높은 원인

  • 네트워크 케이블이나 USB 랜카드가 느슨하게 연결된 경우
  • 네트워크 어댑터 드라이버가 설치되지 않았거나 손상된 경우
  • 메인보드의 내장 랜(LAN) 포트 자체가 고장난 경우
  • USB 허브를 통한 연결에서 전력 부족으로 장치가 제대로 인식되지 않는 경우
  • BIOS에서 내장 랜 기능이 비활성화되어 있는 경우

첫 점검 항목

  1. 케이블과 어댑터 연결 상태를 물리적으로 다시 확인하세요.
  2. 장치 관리자에서 네트워크 어댑터가 목록에 정상적으로 보이는지 확인하세요.
  3. 보이지 않는다면 제조사 홈페이지에서 랜 드라이버를 다시 받아 설치하세요.
  4. USB 랜카드라면 허브를 거치지 않고 메인보드에 직접 연결해 보세요.
  5. BIOS에서 내장 랜(LAN) 컨트롤러가 활성화되어 있는지 확인하세요.

데이터와 시스템을 보호하는 주의사항

시스템 폴더 권한 변경이나 레지스트리 정리 도구 사용보다 저장 공간, 네트워크, 서비스 상태를 먼저 확인하세요.

앞 단계로 해결되지 않을 때

같은 코드가 반복되면 Windows Update 기록과 DISM·SFC 결과를 저장하고 설치 미디어의 버전이 현재 Windows와 맞는지 확인하세요.

복구, 초기화, 드라이버 제거처럼 되돌리기 어려운 작업 전에는 중요한 파일을 다른 물리 저장장치나 신뢰할 수 있는 백업 위치에 보관하세요.

관련 증상 가이드 확인하기

같은 증상과 연결된 다른 오류 코드

  • 0x800704CF · 이 네트워크에 연결되어 있지 않습니다
  • 0x80070718 · IP 주소를 가져올 수 없음
  • 0x80070490 · 요소를 찾을 수 없음 (네트워크 어댑터)
  • 배틀넷 연결 오류 · 배틀넷 오류로 오버워치 등이 실행되지 않음 (BLZBNTBNA0000)

자주 묻는 질문

  • 공유기를 바꾸면 해결되나요? 이 문제(0x80070102)는 공유기 자체 문제보다 어댑터 드라이버나 전원 관리 설정이 원인인 경우가 많아, 장비 교체 전에 드라이버 업데이트와 절전 설정부터 확인해보세요.
  • 공유기를 바꾸면 확실히 해결되나요? 공유기 노후화가 원인이라면 도움이 되지만, PC의 무선랜 드라이버나 절전 설정이 원인이라면 공유기를 바꿔도 재발할 수 있습니다.
