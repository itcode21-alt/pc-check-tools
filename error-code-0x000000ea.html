에러 코드 상세

0x000000EA · THREAD_STUCK_IN_DEVICE_DRIVER

그래픽 드라이버나 디바이스 드라이버가 응답하지 않을 때 연결됩니다.

이 코드를 어떻게 해석해야 하나요?

그래픽 계열 정지는 드라이버 복구 실패뿐 아니라 GPU 온도, 보조전원, 오버클럭, 특정 앱의 하드웨어 가속에서도 발생합니다. 화면이 돌아왔는지와 시스템 전체가 재부팅됐는지를 먼저 구분하세요.

먼저 기록할 내용: 0x000000EA가 발생한 시각, 직전에 실행한 작업, 최근 설치한 드라이버·업데이트, 재부팅 뒤 같은 코드가 반복되는지를 적어 두세요.

가능성 높은 원인

  • GPU 드라이버
  • 과도한 발열
  • 하드웨어 응답 지연

첫 점검 항목

  1. 드라이버 재설치 — 되돌리기 쉬운 항목부터 확인해 위험 없이 범위를 줄입니다.
  2. 온도 확인 — 첫 확인 결과와 비교해 소프트웨어와 하드웨어 가능성을 나눕니다.
  3. 부하 테스트 후 재현 여부 확인 — 앞 단계에서 해결되지 않을 때 다음 원인 후보를 검증합니다.

데이터와 시스템을 보호하는 주의사항

드라이버를 여러 버전으로 연속 덮어쓰기보다 현재 버전과 발생 조건을 기록한 뒤 안정 버전 하나로 비교하세요.

앞 단계로 해결되지 않을 때

화면 깨짐이나 고온이 동반되면 부하 테스트보다 냉각과 전원 연결을 먼저 확인하고, 정상 온도에서도 반복되면 다른 출력 경로와 GPU를 교차합니다.

복구, 초기화, 드라이버 제거처럼 되돌리기 어려운 작업 전에는 중요한 파일을 다른 물리 저장장치나 신뢰할 수 있는 백업 위치에 보관하세요.

관련 증상 가이드 확인하기

같은 증상과 연결된 다른 오류 코드

  • 0x0000001A · MEMORY_MANAGEMENT
  • 0x00000116 · VIDEO_TDR_FAILURE
  • 0x0000009C · MACHINE_CHECK_EXCEPTION
  • 0x00000124 · WHEA_UNCORRECTABLE_ERROR

자주 묻는 질문

  • 게임할 때만 재부팅되면 그래픽카드 문제인가요? 이 문제(0x000000EA)는 그래픽 드라이버 문제일 가능성이 높지만 전원 공급이나 온도 문제와 겹치는 경우도 많아, 드라이버 재설치와 함께 온도·전력 상태도 함께 확인하는 것이 좋습니다.
  • 콘센트나 멀티탭도 원인이 될 수 있나요? 드물지만 불안정한 멀티탭이나 정전압 문제가 원인이 될 수 있어, 다른 콘센트로 테스트해보는 것도 방법입니다.
