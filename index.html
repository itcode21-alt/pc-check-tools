HOW IT WORKS

문제를 세 단계로 좁혀 보세요

보이는 증상부터 시작해 코드와 로그를 차례로 연결하면 원인 후보가 선명해집니다.

증상 선택

가장 가까운 항목을 고르면 원인 후보와 점검 순서를 볼 수 있습니다.

에러 코드 입력

화면에 나온 코드를 넣으면 관련 오류와 함께 보는 항목을 바로 확인할 수 있습니다.

로그 분석

하드웨어 로그를 올리면 점검할 부품과 설정 항목을 우선순위로 정리합니다.

이벤트 기록 확인

이벤트 ID와 원본, 반복 횟수를 함께 보면 비정상 종료·디스크·드라이버 기록을 더 정확히 해석할 수 있습니다.

추천 진단 흐름

증상 → 코드 → 로그 → 이벤트 순서로 좁혀 보세요

먼저 화면에 보이는 증상을 선택하고, 코드가 있으면 함께 검색하세요. 하드웨어 로그와 같은 시각의 이벤트 기록을 연결하면 부품 문제와 Windows 설정 문제를 구분하기가 쉬워집니다.
