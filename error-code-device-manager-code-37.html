에러 코드 상세

코드 37 · 제어 소프트웨어가 업데이트되지 않음

드라이버 버전이 장치와 호환되지 않거나 업데이트가 필요할 때 나타납니다. 코드 37은 드라이버 업데이트의 필요성을 가장 명확하게 나타내는 신호입니다.

이 코드를 어떻게 해석해야 하나요?

Windows가 설치된 드라이버가 장치와 충분히 호환되지 않거나, 더 적합한 드라이버가 있다고 판단했다는 의미입니다. 해결 방법이 명확한 코드이므로, 드라이버 업데이트로 즉시 해결될 가능성이 높습니다.

먼저 기록할 내용: 어느 장치에서 이 오류가 발생하는지, 현재 드라이버 버전이 무엇인지를 확인하세요.

가능성 높은 원인

  • 드라이버가 구형 버전 (가장 일반적)
  • 최신 Windows 버전과의 호환성 문제
  • 새로운 드라이버가 출시됨

첫 점검 항목 (권장순서대로 실행)

  1. 드라이버 자동 업데이트 시도 — 장치를 마우스 오른쪽으로 클릭해 "드라이버 업데이트"를 선택합니다. Windows Update를 통해 최신 드라이버를 자동 검색합니다.
  2. 제조사 공식 드라이버 다운로드 — 자동 업데이트로 해결되지 않으면, PC 또는 부품 제조사 웹사이트에서 해당 모델의 최신 드라이버를 다운로드합니다. 특히 GPU, 오디오 카드, 네트워크 어댑터는 제조사 공식 드라이버가 최신일 가능성이 높습니다.
  3. 드라이버 버전 확인 — 업데이트 후 장치 속성에서 드라이버 버전이 변경되었는지 확인합니다.

드라이버 다운로드 팁

제조사 웹사이트에서 드라이버를 찾을 때 정확한 모델명과 운영체제(Windows 10/11, 32비트/64비트)를 입력해야 합니다. 잘못된 버전을 설치하면 코드 37이 계속 나타날 수 있습니다.

데이터와 시스템을 보호하는 주의사항

드라이버 업데이트 후 시스템이 불안정하면, 이전 버전으로 되돌릴 수 있습니다. 장치를 마우스 오른쪽으로 클릭한 후 "드라이버 버전"에서 "이전 버전 복구" 또는 "드라이버 롤백" 옵션이 있는지 확인하세요.

앞 단계로 해결되지 않을 때

드라이버 업데이트 후에도 코드 37이 계속 나타나면:

  • 드라이버를 제거한 뒤 다시 설치해보기
  • 더 최신의 베타 버전 드라이버가 있는지 확인
  • 장치 제조사 기술 지원에 문의

자주 묻는 질문

  • 코드 37이 나타나면 꼭 업데이트해야 하나요? 네, 이 코드는 드라이버 업데이트가 필요하다는 명확한 신호입니다. 업데이트하지 않으면 호환성 문제로 장치가 완전히 작동하지 않을 수 있습니다.
  • 최신 드라이버 설치 후 문제가 생기면 어떻게 하나요? 드라이버를 이전 버전으로 롤백할 수 있습니다. 장치 속성에서 "이전 버전 복구" 옵션을 찾아보세요.
