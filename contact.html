개인정보는 올리지 마세요

GitHub Issues는 공개 게시판입니다. 이메일, Windows 제품 키, 장치 일련번호, 계정 정보, 로그에 포함된 사용자 이름을 적지 마세요. 개인정보가 포함된 문의는 구글 폼을 이용해 주세요.
