에러 코드 상세

0x000000B4 · VIDEO_DRIVER_INIT_FAILURE

그래픽 드라이버(NVIDIA, AMD, Intel GPU 드라이버)가 초기화에 실패했을 때 발생합니다. 주로 게임 실행 중에 나타납니다.

이 코드를 어떻게 해석해야 하나요?

비디오 드라이버는 부팅 시 초기화되지만, 게임이나 그래픽 프로그램이 시작될 때 추가 초기화를 수행합니다. 이 과정에서 드라이버가 그래픽 카드와 통신하지 못하거나 드라이버 자체에 문제가 있으면 이 오류가 발생합니다.

먼저 기록할 내용: 어느 게임에서 발생하는지, 모든 게임에서 발생하는지, 최근 그래픽 드라이버를 업데이트했는지, 그래픽 카드가 과열되지 않는지를 확인하세요.

가능성 높은 원인

  • 그래픽 드라이버 손상 또는 호환성 문제 (가장 일반적)
  • 그래픽 카드 과열 또는 전원 부족
  • 게임이 요구하는 그래픽 기능을 드라이버가 지원하지 않음
  • 그래픽 카드 하드웨어 오류 (드라이버 모두 재설치해도 계속 발생)

첫 점검 항목

  1. 그래픽 드라이버 최신 버전 설치 — 제조사 공식 웹사이트(NVIDIA, AMD, Intel)에서 최신 드라이버를 다운로드 후 설치합니다.
  2. 이전 버전의 드라이버로 복구 — 최신 버전에서 문제가 생기면 이전 버전(한두 달 전)으로 돌려 봅니다.
  3. 그래픽 카드 온도 확인 — HWiNFO 또는 GPU-Z로 카드 온도를 확인합니다. 80°C 이상이면 먼지 제거가 필요할 수 있습니다.
  4. 파워(PSU) 전원 공급 확인 — 고사양 그래픽 카드는 추가 전원이 필요합니다. 8핀, 6핀 연결이 제대로 되어 있는지 확인하세요.

드라이버 완전 제거 및 재설치

기본적인 드라이버 업데이트로 해결되지 않으면, 드라이버를 완전히 제거한 후 깨끗한 상태에서 다시 설치해야 합니다. NVIDIA 사용자는 DDU(Display Driver Uninstaller), AMD 사용자는 AMD Clean Uninstall Utility를 사용하세요.

특정 게임만 문제가 있는 경우

모든 게임에서 발생하지 않고 특정 게임에서만 발생한다면, 게임 설정에서 그래픽 옵션을 낮춰보세요. 특히 해상도, 텍스처 품질, 레이 트레이싱 등을 낮춰 테스트합니다.

데이터와 시스템을 보호하는 주의사항

드라이버 제거 후 시스템이 불안정할 수 있으므로, 중요한 파일은 미리 백업해두세요. Windows 재설치가 필요할 수도 있습니다.

앞 단계로 해결되지 않을 때

드라이버 재설치 후에도 계속 발생한다면 그래픽 카드 하드웨어 자체에 문제가 있을 가능성이 높습니다. 다른 컴퓨터에서 같은 카드를 테스트해보거나, 그래픽 카드를 교체해야 할 수 있습니다.

게임 중 재부팅 전체 진단 가이드 확인하기

같은 증상과 연결된 다른 오류 코드

  • 0x00000116 · VIDEO_TDR_FAILURE
  • 0x00000117 · VIDEO_TDR_TIMEOUT_DETECTED
  • 코드 43 · Windows에서 이 장치를 중지했습니다

자주 묻는 질문

  • 최신 드라이버를 설치했는데도 계속 오류가 나요. 드라이버를 완전히 제거한 후 재설치해보세요. 부분적으로 설치된 오래된 버전이 남아있을 수 있습니다.
  • 게임마다 다른 오류가 나기도 하는데 이것도 드라이버 때문인가요? 네, 게임마다 그래픽 기능 사용이 다르기 때문입니다. 특정 기능(예: 레이 트레이싱)을 비활성화해보세요.
