에러 코드 상세

0x800704CF · 이 네트워크에 연결되어 있지 않습니다

네트워크 어댑터가 활성화되어 있지 않거나 물리적으로 연결이 끊긴 상태에서 인터넷 관련 작업을 시도했을 때 나타나는 오류입니다. 케이블이 꽂혀 있어도 어댑터가 비활성화되어 있으면 같은 메시지가 나타날 수 있습니다.

이 코드를 어떻게 해석해야 하나요?

업데이트 관련 코드는 네트워크, 캐시, 구성 요소 저장소, 디스크 공간, 서비스 상태 중 어느 지점에서 작업이 중단됐는지를 보여줍니다. 실패가 반복되는 시점과 이벤트 로그의 세부 코드를 함께 확인하면 원인을 좁히기 쉽습니다.

먼저 기록할 내용: 0x800704CF가 발생한 시각, 직전에 실행한 작업, 최근 설치한 드라이버·업데이트, 재부팅 뒤 같은 코드가 반복되는지를 적어 두세요.

가능성 높은 원인

  • 이더넷 케이블이 느슨하게 꽂혀 있거나 손상된 경우
  • 네트워크 어댑터가 장치 관리자에서 비활성화되어 있는 경우
  • 무선 어댑터의 비행기 모드가 켜져 있는 경우
  • 네트워크 드라이버가 손상되었거나 최근 업데이트로 문제가 생긴 경우
  • 공유기나 스위치 포트 자체의 고장

첫 점검 항목

  1. 작업 표시줄의 네트워크 아이콘에서 비행기 모드가 꺼져 있는지 확인하세요.
  2. 장치 관리자에서 네트워크 어댑터가 활성화 상태인지 확인하고, 비활성화되어 있다면 사용으로 전환하세요.
  3. 이더넷 케이블이라면 다른 케이블이나 다른 포트로 교체해 재현 여부를 확인하세요.
  4. 네트워크 드라이버를 제조사 최신 버전으로 재설치해 보세요.
  5. 다른 기기를 같은 케이블/포트에 연결해 공유기나 스위치 쪽 문제인지 구분하세요.

데이터와 시스템을 보호하는 주의사항

시스템 폴더 권한 변경이나 레지스트리 정리 도구 사용보다 저장 공간, 네트워크, 서비스 상태를 먼저 확인하세요.

앞 단계로 해결되지 않을 때

같은 코드가 반복되면 Windows Update 기록과 DISM·SFC 결과를 저장하고 설치 미디어의 버전이 현재 Windows와 맞는지 확인하세요.

복구, 초기화, 드라이버 제거처럼 되돌리기 어려운 작업 전에는 중요한 파일을 다른 물리 저장장치나 신뢰할 수 있는 백업 위치에 보관하세요.

관련 증상 가이드 확인하기

같은 증상과 연결된 다른 오류 코드

  • 0x80070718 · IP 주소를 가져올 수 없음
  • 0x80070490 · 요소를 찾을 수 없음 (네트워크 어댑터)
  • 0x80070102 · 네트워크 장치가 연결되지 않음
  • 배틀넷 연결 오류 · 배틀넷 오류로 오버워치 등이 실행되지 않음 (BLZBNTBNA0000)

자주 묻는 질문

  • 공유기를 바꾸면 해결되나요? 이 문제(0x800704CF)는 공유기 자체 문제보다 어댑터 드라이버나 전원 관리 설정이 원인인 경우가 많아, 장비 교체 전에 드라이버 업데이트와 절전 설정부터 확인해보세요.
  • 공유기를 바꾸면 확실히 해결되나요? 공유기 노후화가 원인이라면 도움이 되지만, PC의 무선랜 드라이버나 절전 설정이 원인이라면 공유기를 바꿔도 재발할 수 있습니다.
