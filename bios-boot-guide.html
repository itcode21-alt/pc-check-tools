SAFE BOOT CHECK

Windows가 안 들어가도, BIOS 업데이트부터 하지 마세요

부팅이 멈추면 먼저 저장장치 인식과 부팅 순서를 확인합니다. BIOS 업데이트는 제조사 안내와 정확한 모델·전원 환경을 갖춘 경우에만 마지막 단계로 진행하세요.

주의

BIOS 업데이트 중 전원이 꺼지면 메인보드가 부팅되지 않을 수 있습니다. 모델이 확실하지 않거나 시스템이 불안정하면 업데이트를 진행하지 말고 제조사 지원을 이용하세요.

1. BIOS 화면 들어가기

전원을 켠 직후 제조사 로고가 보일 때 Del 또는 F2를 여러 번 누릅니다. 노트북은 F1·F10·Esc인 경우도 있어 시작 화면의 안내를 우선 따릅니다.

2. 가장 먼저 볼 두 가지

  1. 저장장치 인식: Main·Storage·NVMe 메뉴에서 Windows가 설치된 SSD/HDD 모델명이 보이는지 확인합니다. 보이지 않으면 전원을 끈 뒤 케이블·슬롯을 점검합니다.
  2. 부팅 순서: Boot 메뉴에서 Windows Boot Manager 또는 Windows가 설치된 SSD를 첫 번째로 둡니다. USB·네트워크 부팅은 임시로 뒤로 보냅니다.

3. 설정을 바꾼 뒤 문제라면

Load Optimized Defaults 또는 Load Default Settings를 선택해 기본값을 불러온 뒤 저장·재시작합니다. XMP/EXPO·오버클럭·RAID·CSM 설정은 현재 Windows 설치 방식과 맞지 않으면 부팅 문제를 만들 수 있으므로, 한 번에 여러 항목을 바꾸지 마세요.

4. BIOS 업데이트가 필요한 경우

  • 메인보드 제조사 지원 페이지에서 정확한 보드 모델과 리비전을 확인합니다.
  • 해당 모델의 변경 이력에 현재 증상 또는 CPU·메모리 호환성 개선이 있는지 확인합니다.
  • 제조사 설명서의 전용 업데이트 기능만 사용하고, 안정적인 전원에서 진행합니다.
  • 업데이트 전 현재 BIOS 버전과 설정을 사진으로 남깁니다.
