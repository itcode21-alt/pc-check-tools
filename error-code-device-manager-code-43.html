에러 코드 상세

코드 43 · Windows에서 이 장치를 중지했습니다 (코드 43)

드라이버가 장치에서 하드웨어 오류나 문제를 감지해 윈도우에 보고했을 때 나타나는 코드로, 그래픽카드나 USB 장치에서 특히 자주 나타납니다. 소프트웨어 문제일 수도 있지만 하드웨어 결함의 신호인 경우도 많습니다.

이 코드를 어떻게 해석해야 하나요?

업데이트 관련 코드는 네트워크, 캐시, 구성 요소 저장소, 디스크 공간, 서비스 상태 중 어느 지점에서 작업이 중단됐는지를 보여줍니다. 실패가 반복되는 시점과 이벤트 로그의 세부 코드를 함께 확인하면 원인을 좁히기 쉽습니다.

먼저 기록할 내용: 코드 43가 발생한 시각, 직전에 실행한 작업, 최근 설치한 드라이버·업데이트, 재부팅 뒤 같은 코드가 반복되는지를 적어 두세요.

가능성 높은 원인

  • 그래픽카드가 과열되거나 전원 공급이 불안정한 경우
  • USB 장치가 순간적으로 과전류를 발생시켜 컨트롤러가 안전을 위해 차단한 경우
  • 드라이버 버그로 장치 초기화에 실패하는 경우
  • 펌웨어가 오래되어 최신 드라이버와 호환되지 않는 경우
  • 하드웨어 자체의 결함(그래픽카드 고장, USB 컨트롤러 손상 등)

첫 점검 항목

  1. 장치를 다른 포트나 슬롯에 연결해 재현되는지 확인하세요.
  2. 그래픽카드라면 드라이버를 완전히 제거(DDU 등)한 뒤 최신 버전으로 재설치하세요.
  3. USB 장치라면 다른 PC에서도 같은 증상이 나오는지 확인해 장치 자체 결함인지 구분하세요.
  4. 온도와 전원 공급 상태를 점검하세요 (특히 그래픽카드 보조 전원).
  5. 제조사 펌웨어 업데이트가 있는지 확인하고 적용해 보세요.

실제 사용자 사례

위 점검 순서로도 해결되지 않았을 때, 다른 원인으로 해결된 사례입니다.

파워서플라이 교체로 해결한 사례

RTX 2070에서 코드 43이 발생해 드라이버 재설치, DDU 클린 설치, PCIe 슬롯 교체까지 시도했지만 해결되지 않았던 사례가 있습니다. 자가 진단 도구로는 파워서플라이가 정상으로 나왔지만, 커뮤니티에서는 오래되고 품질이 낮은 파워서플라이 자체를 의심해보라고 권했고, 실제로 파워서플라이를 교체한 뒤 문제가 해결됐습니다.

포인트: 전원 공급 장치는 테스트 도구에서 이상 없다고 나와도 실제로는 불안정한 경우가 있습니다.

램 슬롯 분리 테스트로 해결한 사례

RMA로 교체받은 그래픽카드에서 코드 43과 함께 화면에 줄이 가는 증상이 나타났는데, 커뮤니티에서는 그래픽카드 자체보다 램(RAM) 문제일 가능성을 짚었습니다. 서로 다른 시기에 구매한 메모리 키트를 함께 꽂아 쓰고 있었던 것이 원인일 수 있다는 조언에 따라 메모리 키트를 하나씩 분리해서 테스트하는 방법이 제안됐습니다.

포인트: 그래픽 관련 오류라고 해서 반드시 그래픽카드만의 문제는 아닐 수 있습니다.

데이터와 시스템을 보호하는 주의사항

시스템 폴더 권한 변경이나 레지스트리 정리 도구 사용보다 저장 공간, 네트워크, 서비스 상태를 먼저 확인하세요.

앞 단계로 해결되지 않을 때

같은 코드가 반복되면 Windows Update 기록과 DISM·SFC 결과를 저장하고 설치 미디어의 버전이 현재 Windows와 맞는지 확인하세요.

복구, 초기화, 드라이버 제거처럼 되돌리기 어려운 작업 전에는 중요한 파일을 다른 물리 저장장치나 신뢰할 수 있는 백업 위치에 보관하세요.

관련 증상 가이드 확인하기

같은 증상과 연결된 다른 오류 코드

  • 0x0000001A · MEMORY_MANAGEMENT
  • 0x00000116 · VIDEO_TDR_FAILURE
  • 0x000000EA · THREAD_STUCK_IN_DEVICE_DRIVER
  • 0x0000009C · MACHINE_CHECK_EXCEPTION

자주 묻는 질문

  • 게임할 때만 재부팅되면 그래픽카드 문제인가요? 이 문제(코드 43)는 그래픽 드라이버 문제일 가능성이 높지만 전원 공급이나 온도 문제와 겹치는 경우도 많아, 드라이버 재설치와 함께 온도·전력 상태도 함께 확인하는 것이 좋습니다.
  • 콘센트나 멀티탭도 원인이 될 수 있나요? 드물지만 불안정한 멀티탭이나 정전압 문제가 원인이 될 수 있어, 다른 콘센트로 테스트해보는 것도 방법입니다.
