매달 패치 데이(둘째 주 화요일) 직후, Microsoft가 공식적으로 확인한 KB 업데이트 이슈를 조사중·워크어라운드·해결 상태와 함께 정리합니다.

여기 정리하는 내용은 추측이나 커뮤니티 소문이 아니라 Microsoft의 공식 릴리스 상태 문서(Release Health)에 실제로 등재된 이슈만 다룹니다. 이 사이트의 다른 진단 도구와 마찬가지로 서버로 데이터를 전송하지 않고 브라우저에서 바로 확인할 수 있다는 점이 강점입니다.

업데이트 주기: Microsoft가 새 KB를 배포하는 매달 둘째 주 화요일(패치 데이) 이후 며칠 내로 이 페이지를 갱신합니다.

이번 달 이슈 (2026년 7월)

KB5101650

2026-07-14 · Windows 11
조사중 워크어라운드 있음

이 업데이트 이후 보고된 주요 이슈 4가지입니다.

  • 일부 Dell·Intel 프로세서 기기 — 하드웨어 호환성 문제로 예기치 않은 종료, 성능 저하, 발열 증가, 배터리 소모가 발생할 수 있습니다. Microsoft와 Dell이 공동으로 원인을 조사 중이며, 해당하는 일부 기기에는 이 업데이트가 자동으로 제공되지 않습니다.
  • 키보드 단축키 무응답 — 단축키 해제·정리 동작이 변경되면서, 일부 내장 Windows 기능이 특정 단축키에 반응하지 않을 수 있습니다. 해당 앱을 재시작하면 대부분 해결됩니다.
  • 서드파티 앱의 소켓 통신 중단 — 보안 강화로 TDI 트랜스포트 등록 요건이 강제되어, 등록되지 않은 서드파티 TDI 트랜스포트로 소켓 통신을 하던 앱이 동작을 멈출 수 있습니다. 정식 등록된 트랜스포트는 영향이 없습니다.
  • Office 자동화 앱 실행 실패 — OLE Automation으로 Office를 여는 일부 서드파티 앱이, 2026년 6월 보안 업데이트 이후 Office 실행이나 문서 열기에 실패하는 문제가 함께 보고되었습니다.

출처: Microsoft Support: July 14, 2026—KB5101650

KB5094126

2026-06-09 · Windows 11
해결됨 (KB5101650에서 수정)

파일을 완전 삭제할 때 뜨는 확인 대화상자에, 원래 파일 이름 대신 휴지통 내부 파일명이 표시되는 문제가 있었습니다. 실제 삭제 동작에는 영향이 없었지만 사용자가 다른 파일을 지우는 것으로 착각할 수 있는 표시 오류였습니다. 2026년 7월 14일 업데이트(KB5101650)로 수정되었습니다.

출처: Microsoft Support: June 9, 2026—KB5094126

지난 이슈

KB5083769

2026-04-14
워크어라운드 있음

취약한 커널 드라이버를 차단하는 보안 강화(vulnerable driver blocklist)가 적용되면서, 해당 드라이버에 의존하는 일부 백업 프로그램이 디스크 이미지를 마운트하거나 관리할 때 실패할 수 있습니다. 사용 중인 백업 소프트웨어가 영향을 받는지 제조사 공지를 먼저 확인하세요.

출처: Microsoft Support: April 14, 2026—KB5083769

이 트래커 보는 법

  • 조사중 — Microsoft가 이슈를 확인했지만 아직 원인 조사나 수정 작업이 진행 중인 상태입니다.
  • 워크어라운드 있음 — 정식 수정 전이지만 우회 방법이나 조건부 회피 방법이 공식적으로 안내된 상태입니다.
  • 해결됨 — 후속 업데이트로 정식 수정되었습니다. 여전히 증상이 남아있다면 해당 후속 업데이트가 설치되어 있는지 먼저 확인하세요.

이 페이지는 특정 KB 하나를 단독 페이지로 다루지 않고, 매달 새 이슈를 이 타임라인에 이어서 정리하는 방식으로 운영합니다. 특정 KB에 대한 검색 수요가 뚜렷하게 확인되면 별도 상세 페이지로 분리할 수 있습니다.

자주 묻는 질문

  • 알려진 이슈가 있으면 업데이트를 미뤄야 하나요? 보안 업데이트 자체를 미루는 것은 권장하지 않습니다. 다만 특정 하드웨어·소프트웨어 조합에서만 발생하는 이슈라면, 본인 환경이 해당하는지 먼저 확인하고 워크어라운드가 있는지 살펴본 뒤 설치하는 것이 안전합니다.
  • 이 트래커에 없는 문제를 겪고 있다면 어떻게 하나요? 이 페이지는 Microsoft가 공식적으로 확인한 이슈 위주로 정리합니다. 여기 없는 증상이라면 진단 도구에서 오류 코드나 이벤트 ID로 먼저 검색해 보시고, 그래도 원인을 못 찾으면 문의를 통해 알려주세요.
  • '해결됨' 표시는 무엇을 의미하나요? Microsoft가 후속 업데이트로 문제를 수정했다고 공식 발표한 경우입니다. 다만 일부 환경에서는 이전 업데이트가 이미 설치되어 있어야 수정이 적용되므로, 여전히 증상이 남아있다면 최신 누적 업데이트 설치 여부를 확인하세요.
